JioTV Direct

Latest Release: v1.1.3

Official Repository for JioTV Direct Kodi Addon.

Download Repository ZIP

Kodi Source URL

Add this URL to your Kodi File Manager:

https://dineshintry.github.io/plugin.kodi.jiotv/

Installation Steps

  1. In Kodi, go to Settings > File Manager > Add Source.
  2. Enter the URL above and name it JioTV Direct Repo.
  3. Go to Settings > Add-ons > Install from zip file.
  4. Select JioTV Direct Repo and choose repository.jiotvdirect-1.0.0.zip.
  5. Once installed, select Install from repository > JioTV Direct Repository > Video add-ons > JioTV Direct.
